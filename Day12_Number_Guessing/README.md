# 🎯 Number Guessing Game (CLI)

A robust, command-line based Number Guessing Game built with Python. This project demonstrates foundational programming concepts, strict scope management, and the implementation of an AI Agentic Workflow for code optimization.

## 🚀 Features
* **Two Difficulty Levels:** Easy (10 turns) and Hard (5 turns).
* **Input Validation:** Prevents application crashes if the user inputs strings or invalid characters using `try-except` blocks.
* **Strict Scoping:** Avoids global variable mutation, encapsulating the game logic cleanly within the main execution block.
* **Type Hinting:** Fully type-hinted functions (PEP 8 compliant) for better readability and maintainability.

## 🧠 The "AI Agentic Workflow" Approach
This project was developed using a modern, multi-agent approach to simulate a real-world software development lifecycle:
1. **Base Logic (Human):** Core algorithm, loops, and conditional logic written manually.
2. **Code Review Agent (AI):** Analyzed the base code for scope pollution and Big-O inefficiencies.
3. **Refactoring Agent (AI):** Restructured the code for modularity, adding Type Hints and `ValueError` exception handling.
4. **Documentation Agent (AI):** Generated this professional README to highlight the development pipeline.
